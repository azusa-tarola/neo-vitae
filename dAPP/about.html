How it works

Usage

Using this website you can "search" for certifications given to people that are stored on the NEO blockchain. If you know the wallet address of that person you can check all the issued documents to them.

With this scheme you can trust the guarantees provided by the blockchain and you do not need to manually verify the validity of the certifications or even if they were forged.

As a person being certified you no longer need to rely on paper documents to prove your past experience.

As a certification entity you only need to create a "json" document add it to the IPFS network and run our smart-contract. For more information about how to issue certificates can be found here

Technology

To achieve the desired functionality we make use of 2 decentralized technologies, NEO and IPFS, so you do not need to rely on a central server.

More information can be found on the documentation section of the code repository.
